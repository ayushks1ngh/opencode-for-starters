# Roadmap: Research Agent SDK

## Phase 1: MVP (AC-1 through AC-6)
Core abstractions with mock provider and basic examples.
- Tool abstraction + ToolRegistry (AC-2)
- Memory abstraction + InMemoryMemory (AC-3)
- Provider abstraction + MockProvider (AC-4)
- Agent abstraction (AC-1)
- Workflow abstraction (AC-5)
- Basic examples (AC-6)

## Phase 2: Real Providers & Persistence
- OpenAI provider implementation
- Anthropic provider implementation
- File-based memory persistence
- Async support

## Phase 3: Advanced Features
- Tool parameter schemas (Pydantic-inspired)
- Streaming support
- Agent delegation (sub-agents)
- Built-in tool set (web search, calculator, file read)

## Phase Mapping
- Phase 1: AC-1, AC-2, AC-3, AC-4, AC-5, AC-6
- Phase 2: (future ACs)
- Phase 3: (future ACs)

## Effort Estimates
- Phase 1: ~2-3 hours implementation
- Phase 2: ~3-4 hours
- Phase 3: ~4-5 hours
